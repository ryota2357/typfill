# Typfill

> 🚧 Work in progress

A web app that generates PDFs from Typst templates entirely in the browser. Nothing is sent to a server — compilation runs in-browser via WebAssembly.

## Templates

- Resume (履歴書)
- Invoice (請求書)

## Stack

SvelteKit (static adapter) / TypeScript / Tailwind CSS / [typst.ts](https://github.com/Myriad-Dreamin/typst.ts)
